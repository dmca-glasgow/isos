Supported values for `\theoremstyle` are `plain`, `definition` and `remark`:

<CodeSnippet
  latex={String.raw`

\documentclass{article}
\usepackage{amsthm}

\theoremstyle{plain}
\newtheorem{theorem}{Theorem}

\theoremstyle{definition}
\newtheorem{lemma}[theorem]{Lemma}

\theoremstyle{remark}
\newtheorem{corollary}[theorem]{Corollary}

\begin{document}

\begin{theorem}[Theorem name] Lorem ipsum dolor sit amet, consectetur adipiscing elit. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit. \end{theorem}

\begin{corollary}[Theorem name] Lorem ipsum dolor sit amet, consectetur adipiscing elit. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit. \end{corollary}

To reference a theorem use `\cref` and the correct prefix for the theorem:

<CodeSnippet latex={String.raw`
\documentclass{article}
\usepackage{amsthm}
\usepackage[noabbrev, capitalise, nameinlink]{cleveref}
\newtheorem{theorem}{Theorem}[section]
\begin{document}

\section{My Section}

\begin{theorem}
\label{thm:example}
My convention
\end{theorem}

See~\cref{thm:example}.

\end{document}
`} />

Here is a table of the default theorems and their prefixes:

| Prefix | Label       |
| :----- | :---------- |
| thm    | Theorem     |
| lem    | Lemma       |
| cor    | Corollary   |
| prp    | Proposition |
| cnj    | Conjecture  |
| def    | Definition  |
| exm    | Example     |
| exr    | Exercise    |
| sol    | Solution    |
| rem    | Remark      |

Custom theorems can be defined, and their labels will be picked up automatically from the `\newtheorem` definition:

<CodeSnippet latex={String.raw`
\documentclass{article}
\usepackage{amsthm}
\usepackage[noabbrev, capitalise, nameinlink]{cleveref}
\newtheorem{convention}{Convention}
\begin{document}

\begin{convention}[My name]
\label{convention:example}
My convention
\end{convention}

See \cref{convention:example}.

\end{document}
`} />
