### With LaTeX

To include other `.tex` files in your `.tex` file, the `\input` or `\include` macros are supported with recursion (included files can also include files, and so on). In ISOS, both macros behave similarly, where their content is simple "pasted" into the parent document at the position of the macro.

For example, with the following files co-located, `other1.tex`:

<SyntaxHighlight
  language="latex"
  code={String.raw`
    \section{Other 1}

    A paragraph from other1.tex.

    Another paragraph from other1.tex.

    \input{other2.tex}

`}
/>

And `other2.tex`:

<SyntaxHighlight
  language="latex"
  code={String.raw`
    \section{Other 2}

    A paragraph from other2.tex.

`}
/>

Outputs:

<MockCodeSnippet

`}

</section>
<article>
`} />

### With Markdown

Similarly, you can include other files inside `.qmd` or `.md` files by using the macro `{{< include x >}}`.

For example, with the following files co-located, `other1.qmd`:

<SyntaxHighlight
  language="md"
  code={String.raw`
    ## Other 1

    A paragraph from other1.qmd.

    Another paragraph from other1.qmd.

    {{< include other2.qmd >}}

`}
/>

`other2.qmd`:

<SyntaxHighlight
  language="md"
  code={String.raw`
    ## Other 2

    Text from other2.qmd.

`}
/>

And `demo.py`:

<SyntaxHighlight
  language="python"
  code={String.raw`
    # This program prints Hello, world!
    print('Hello, world!')
`}
/>

With the following output:
